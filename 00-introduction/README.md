## ğŸ“š What is Vue.js?

Vue.js is a progressive JavaScript framework for building user interfaces. It's:
- **Easy to learn**: Especially if you know HTML, CSS, and JavaScript
- **Flexible**: Works for small widgets or large applications
- **Reactive**: Automatically updates the UI when data changes
- **Component-based**: Build UIs from reusable pieces

## â“ Common Questions

**Q: Can I use Vue 2 instead of Vue 3?**  
A: These exercises are designed for Vue 3 (Composition API). Vue 3 is the current standard.

**Q: Do I need to know TypeScript?**  
A: No, we'll use JavaScript. TypeScript is optional.

**Q: How long should each exercise take?**  
A: Early exercises: 1-2 hours. Later exercises: 2-4 hours. Mini project: 8-12 hours.

**Q: Can I work in groups?**  
A: Discuss concepts together, but write your own code individually.

**Q: What if I get stuck?**  
A: 1) Read the error message, 2) Check Vue docs, 3) Use console.log, 4) Ask classmates/instructor.
