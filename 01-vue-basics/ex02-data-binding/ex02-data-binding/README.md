![Exercise Image](image-exo-02.png)

1. What's the difference between v-bind:class and regular class?
v-bind:class allows us to have dynamic classes based on the bound variable, while regular class is static.
2. When would you use :style vs CSS classes?
:style is for changing inline styles dynamically (like user colors), while classes are for predefined styles.
3. Why is v-model considered two-way binding?
Because if we change the value in the template it updates the script, and if we change the script it updates the template.
